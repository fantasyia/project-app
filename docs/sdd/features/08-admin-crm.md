# Feature Spec: Admin CRM

## 1. Identificacao
- Nome da feature: Admin CRM (KYC, moderacao e operacao financeira)
- Dominio:
  `admin`
- Status:
  `done`

> Update 2026-04-21: a primeira passada completa mobile-first do Admin CRM foi concluida na Fase 2.

## 2. Escopo Entregue
- Shell admin mobile-only (sem sidebar desktop).
- Overview operacional em cards.
- Usuarios em cards mobile.
- Superficies de KYC, moderacao, reembolsos e chargebacks em linguagem de operacao.
- Placeholders de financeiro e ajustes alinhados ao dominio.

## 3. Acceptance Criteria
- [x] Visao de KYC operacional no painel.
- [x] Visao de moderacao operacional no painel.
- [x] Fluxo admin mobile-first sem dependencia de layout desktop.
- [x] RBAC admin preservado.

## 4. Backlog Futuro (Pre-Lancamento)
- [ ] Regras finais de antifraude e automacoes de risco.
- [ ] Integracao com conciliacao real de gateway.
